# 大型单页应用（响应式布局）

本人独立使用vue制作的一个响应式的大型单页应用[在线预览](https://sanmer0312.github.io/WDxitie/dist/index.html#/home)，如果无法打开预览，原因是部分运营商DNS污染（域名指往不正确的IP地址），可以通过 修改hosts文件 / 修改DNS服务器 / 代理 的方式访问
